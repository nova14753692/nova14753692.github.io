Game Development Project

Selena's Revenge

This is an RPG game made with Unity, where player controls a character named Selena to help her find out her father whereabout and eventually save the world.

My team, including me and 3 other members, made this project as an class project for Game Development class within 3 months, in our school. I had other Unity projects but this is my most completed and feature rich game project so far.

As the time of making this game, I had 2 years of Unity experience as a hobbyist, the rest of my team were just get started to know Unity, so it was easy task to convince them to vote for me to be their team lead. :D

However, they learned incredibly fast, and that was a good thing, we needed all the talents for this project, and I also learned a lot from them.

Over the process of making this game, we encoutered many problems arised such as the lacking of 3d artist, even when we had 3d models, there also some problems to intergrate animations and have them play smoothly, there were also programming design problems, so there were times when we had to rollback some commits or features with the help of git and github, and re-designed our code structures as well as business logic. In addition to that, when the deadline was near, there came optimization issues. We needed to have our game run on low-end devices also. We did alot of things to optimized the game including code refactor, adjusted lightning rendering, and made up a lazy engine to only play animations, render graphic, and run code when the objects within some distance. 'Optimization was a huge effort to double the framerates or even triple the framerates on high-end devices.

Finally, after all the effort, the result was great. Our game was the top compare to many other teams in class in term of techniques use in development and gameplay.

Online Shopping App

The app is a craglist-like online shopping app, where user can upload picture of their product, post a price, description, and their location. This time, I look into something really interesting, Flutter framework for cross-platform mobile app development using Dart as programming language.

A bit about Flutter, it is a cross-platform framework to develop app for both Android and iOS. Flutter is introduced by Google around 2 years ago, and current has release version 1. It also has a lot of traction in mobile app development community. Flutter is very easy to learn, and has trememdously fast development speed, even faster when compare to React Native. When talking about faster than React Native, Flutter's performance theoretically extremely close to native code, because instead of maintaining a JavaScript Bridge like React Native, Flutter app compile directly to C/C++ library. For UI Flutter doesn't compile to native UI elements like React Native, it provides proprietary UI widgets that then will be drawn directly to the screen by using Skia graphic engine (the same engine used by Google Chrome). Flutter app will look and feel the same no matter what device user is using, or what OS version the device is running.

I made this app within about a week by following a Udemy course. The app involves many technologies and API from Google Firebase, Google Cloud Storage, and Google Geo-location.

I used Google Firebase to handle account registering, login, authentication, and interaction between the app and server. Google Cloud Storage is used to store products' pictures and Google Geo-location is used to get user exact location, then show a static map using Google Geo-location API. Users can either enter their address, then the app will correct it using Google Geo-location or they can just let the app detect their location.
